Ramaze is working correctly with this application, now you can start working.

To start you can modify:

  1. view/index.xhtml, this page.
  2. view/page.xhtml, the layout for this page.
  3. controller/main.rb, where you can change the header of this page and the notemplate action

For more documentation, check out ramaze.net and wiki.ramaze.net, which includes:

  • a getting started guide
  • some screencasts
  • and an introductory tutorial

You can also read the rdocs or browse around the Ramaze source code.

For help with Ramaze, visit #ramaze on irc.freenode.net.
You can use Mibbit, an AJAX based IRC client or the official freenode irc java applet.
Feel free to post to the Ramaze Google Group, your first mail has to go through moderation, so please be patient.
